## Описание работы
Целью данной работы является разработка программного инструмента на языке Python для автоматизированного сбора данных об угрозах, их последующего анализа и имитации мер оперативного реагирования. Скрипт предназначен для обработки событий сетевой безопасности и верификации подозрительной активности через внешние сервисы.

## Основные этапы реализации
1. **Сбор данных:** Реализовано взаимодействие с двумя источниками – локальными логами Suricata (формат eve.json) и API VirusTotal для проверки репутации IP-адресов.
2. **Анализ данных:** Скрипт анализирует полученные данные, подсчитывает количество алертов для каждого источника и определяет степень угрозы на основе заданных порогов и данных API.
3. **Реагирование на угрозы:** При обнаружении опасности скрипт выполняет имитацию блокировки IP-адреса и отправляет уведомление в Telegram.
4. **Формирование отчетности:** Результаты анализа сохраняются в форматах .json и .csv. Также осуществляется визуализация данных в виде графика распределения угроз в формате .png.

## Технические требования
* Язык разработки: Python 3.8+
* Используемые библиотеки: requests, pandas, matplotlib, json

## Инструкция по использованию
1. Установите необходимые библиотеки с помощью команды:
   `pip install requests pandas matplotlib pytz`
2. Разместите файл логов `eve.json` в корневой директории проекта.
3. Для корректной работы системы уведомлений и API-проверок укажите следующие переменные окружения:
   - `VT_API_KEY` – ключ доступа к VirusTotal API.
   - `TELEGRAM_BOT_TOKEN` – токен Telegram-бота.
   - `TELEGRAM_CHAT_ID` – ID чата для получения уведомлений.
4. Запустите скрипт. В случае отсутствия файла логов программа предложит запустить демонстрационный режим.
